
# Rockeat Movies

This is backend da aplication Rockeat Movies in stage 08 of Rockeatseat explore.




## Required

- O projeto está rodando na versão do nodejs 18.18.0.

## Instalação

Instale Rockeat Movies com npm

```bash
  npm install 
  npm run dev
```
    
## Funcionalidades

- Middleware
- Autenticação em JWT
- Preview em tempo real
- Modo tela cheia
-
- Multiplataforma
